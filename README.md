# 🤖 What is Prompt Engineering?

Prompt Engineering is the process of creating well-structured instructions for AI tools to generate accurate and high-quality outputs.

It involves:

- Giving proper task context  
- Defining audience  
- Setting tone/style  
- Adding constraints  
- Specifying output format  

This helps produce better AI-generated content.

---

# 🧠 Types of Prompt Engineering Used

## 1️⃣ Zero-Shot Prompting
Giving direct instructions without examples.

Example:
"Generate SEO keywords for a restaurant."

---

## 2️⃣ Few-Shot Prompting
Providing examples before generating outputs.

---

## 3️⃣ Instruction-Based Prompting
Giving clear step-by-step requirements.

---

## 4️⃣ Constraint-Based Prompting
Adding limits like:
- word count
- SEO optimization
- location-based targeting

---

## 5️⃣ Iterative Prompting
Improving prompts step-by-step for better results.
